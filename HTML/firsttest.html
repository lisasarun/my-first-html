2.html

tag in HTML is used to denote text in an alternate voice or mood, often displayed in italics.
It's commonly used to represent technical terms, phrases from other languages, ship names, or thoughts.
While traditionally rendered in italics, the tag's primary function is now semantic,
indicating a specific type of text rather than just visual styling.

3.head

In HTML, the tag is a container for metadata,
which is data about the HTML document itself, rather than the content displayed on the page.
It's located between the and tags.
The section doesn't show up in the rendered webpage but is crucial for how the browser and search engines understand and process the page.
